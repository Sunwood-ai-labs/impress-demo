# draw-io-skill 3D Readable Slide

`draw-io-skill` v0.1.0 を紹介する `impress.js` スライドの、3D 遷移と可読性改善を両立したバージョンです。

## 使い方

リポジトリのルートでサーバーを起動します。

```bash
npm install
npm start
```

ブラウザで `http://127.0.0.1:4173/slide/draw-io-skill-3d-readable/` を開くとスライドを確認できます。

## チェック

```bash
npm run check:slide-text:readable
```

## 操作

- `Space`
- `←` / `→`
- `Tab`
